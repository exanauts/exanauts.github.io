ProxAL

ProxAL.jl is a Julia package to solve multiperiod contingency-constrained AC Optimal Power Flow (ACOPF) problems. Its main feature is a distributed parallel implementation, which allows running on high-performance computing architectures.

This document describes the algorithm, API and main functions of ProxAL.jl.
